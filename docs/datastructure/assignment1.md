# [Assignment #1] 다항식의 곱셈 계산

## 개요
교재에 있는 다항식의 합 `polynomial2.c`의 `poly_add2()` 함수의 자료구조를 참고하여, 2개의 다항식의 곱 `multPoly()` 함수를 작성한다.

---

## 구현 조건

1. 다항식의 변수는 다양한 표현이 가능할 것 (예: x, y, xyz 등 변수 세 개까지의 항을 포함)
2. 제곱은 `^` 으로 표현할 것
3. 표현: `"3y^7"` 은 `"+0000003y^7"` 과 같은 표현임
4. 최종 결과물 출력 시 같은 차수 변수는 계산된 한 개만 출력할 것
5. `x^0`, `y^0`, `z^0` 은 상수항으로 처리되도록 표현할 것
6. 헤더 파일은 `#include <stdio.h>`, `#include <stdlib.h>`, `#include <math.h>` 세 개만 이용할 것
7. `scanf` 는 사용 금지 (사용 시 0점 처리)
8. 정렬은 C언어에 내장되어 있는 `qsort` 함수를 이용한다 (최종 다항식은 변수 y에 대하여 내림차순으로 정렬한다)

---

## 실행 예

### `./multPoly` 프로그램 실행 시 4가지 test case에 대해 `printf` 함수를 이용하여 각각의 정답이 출력되도록 구현한다.

### 1) test case (1점)
- A: `3y^7 + 4x^4 - 1`
- B: `5x^4 - 3y^2 + 7`

### 2) general case (3점)
- A: `4y^7 - 4x^4 - 1`
- B: `2y^7 - 3x + 1`

### 3) extreme case (3점)
- A: `15x^4 * y^7 - 9y^9 + 21y^7 + 20x^8`
- B: `-12x^4 * y^2 + 23x^4 + 3y^2 - 7`

### 4) extreme case2 (3점)
- A: `15x^4 * y^7 * 6z^3 - 9y^9 * z^2 + 21y^7 + 20x^8`
- B: `-12x^4 * y^2 * z^2 + 23x^4 + 3y^2 - 7`

---

## 프로그램 작성 환경 조건

1. 학과 리눅스 서버 vi 에디터를 이용하여 작성한 후 turnin에 제출한다.
2. gcc 컴파일러를 이용하여 컴파일한다.

---

## 주의 사항

1. 미 제출 시 0점 처리, 부정행위 적발 시 -10점 처리 (추후 제출하는 숙제에서 감점 처리)

---

## 채점 기준

1. 정렬 안 함 -1점, `qsort` 함수 이용 안 함 -1점, x, z가 y보다 먼저 나오게 정렬 -3점
2. 헤더 파일 추가 이용 -1점
3. 곱셈 중간에 있는 계수를 앞으로 빼지 않고, 안에서 계산되어 최종 결과에 나오도록 한다 -3점
   - 예) `15x^4 * y^7 * 6z^3` → 앞의 계수를 90으로 하지 말고 안에서 곱셈이 일어나도록
4. vi, gcc 컴파일러 (실습 환경) 에서 동작 안 함 → 0점 처리
5. `x^0`, `y^0`, `z^0` 상수항으로 처리 안 함 -1점
6. cpp 파일로 제출 시 -3점
7. 분반 착각 시 -3점
8. 지연 제출은 받지 않음
9. 파일을 여러 개 제출할 경우 제일 낮은 점수의 파일로 채점
10. 카피 적발 -10점 (0점 미만일 경우 누적점수 -점수)
